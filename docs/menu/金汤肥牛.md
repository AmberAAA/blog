# 金汤肥牛

[[toc]]

## 原料

### 必须
1. 南瓜
2. 肥牛
3. 金针菇
4. 青笋
5. 酸菜
6. 海南灯笼椒酱
7. 白醋

### 推荐
1. 豆芽
2. 海带

## 步骤
### 准备工作
1. 把南瓜蒸熟，半个即可，蒸熟后，去掉外面的绿皮，不然会影响汤底颜色。
2. 将你喜欢的配菜焯一下水，肥牛焯水后，再过一下冷水去除一下血沫，

::: tip 提示
1. 酸菜可以不焯水
2. 自己喜欢吃什么菜，就下什么菜。
:::

### 熬制汤底
1. 炒制底料(葱、姜、黄灯笼椒)，出香味后，之后加水约两大碗水。
2. 加入蒸熟的南瓜，并用锅铲压碎，并不断搅拌。
3. 加入食盐、白醋调味。
4. 等沸腾后，再改小火，熬制10分钟。
5. 用细滤网过滤掉，调料渣以及南瓜泥。
6. 出锅，倒入容器中。


### 拼盘
1. 砂锅底部盛放蔬菜，均匀盖上肥牛片。
2. 倒入煎好的汤底，与肥胖平齐即可。
3. 煮沸。
4. 出锅前，放入一团香菜，10颗花椒，浇上热油。
