# 职业
### 职业发展

将今年的事业生涯卖给了极目云，在公司也刚刚敲钟满1年了。去年12月5日入职，经历了职业生涯的第一次出差，到后续的陆陆续续，跑过了河北、河南、北京、山西。到时没有感觉到收获很多。

极目云在今年上半年被天健收购，当初说好的绩效也在发了几次后了无音讯。

极目云是一家典型的软件公司，而我们部门——医联体事业部，却处于一个偏远地带，团代有JAVA两名、.NAT一名、UI一名，已经前端我一个。年初需求还比较明确，项目在不断的迭代开发。到了5月份左右，开始开拓市场，我们也开始频繁出差。但陆陆续续跑了多半年，项目到现在也没有实际使用起来。

今年10月上旬在艾斯广场附近开通了健身房会员卡，现在依然在坚持着锻炼。就算出差不能去的日子，也基本上可以保持在50%左右的出勤率。虽然这一两个月体重也仅仅只降了5斤左右，但我依然能感觉到身体的变化，体检结果（转氨酶现在已经降低到70）也告诉我，我的坚持没有白费。为了保护膝盖，现在在健身房也不怎么跑步，而是采取了快走+爬坡的方式。
