# 2020年年终总结

## 黑天鹅

2020年最大的黑天鹅，大概就是新冠疫情。年初武汉封城，全国戒严。每天新增几万的感染，买不到口罩，全国各地都缺医疗物资。

最初，也是最严重的那几天，也碰巧是在过年。社区发放通行证，强行限制人员流通，每户只允许一人几天内出去买菜。

疫情爆发后，很快就推出了健康码，直到现在，出入公共场所，坐地铁都要扫一扫健康码。

再后来，产能逐渐恢复，疫情也慢慢得到控制，人们开始复工复产。

武汉疫情过去后，北京、新疆等地也小规模爆发过疫情。

本以为疫情会来的快，去的也开，可这是对中国而言。紧接着，韩国、意大利等也先后爆发了疫情。英国最先提出了群体免疫。之后美国的感染人数遥遥领先，懂王川普也得了新冠，但很快就出院。

## 这一年

这一年，买了人生中第一辆车，因为疫情，和萌萌、林林、仲圆一起去北京提车，从中午12点，到次日凌晨2点，长途驱车14小时，开回了西安。

这一年，我和萌萌结婚了，嘉城也装修完毕了。

这一年，换了好几份工作，从陆运帮离职后，有一个多月没有工作，一直在忙装修的事情，9月19日入职华为OD，次年1月31日离职。

这一年，我和萌萌举行了婚礼，婚后我们过的很幸福。

在华为OD在西南三环，每天通勤3小时，往返50公里。再加上紧张的工作节奏，使我身心俱疲，这期间体重也一直在疯涨，离职时，体重已经增长到183斤。

这一年，爸爸一直在吃降压药，妈妈睡眠不好。


## 成长
这一年，有何成长？

时间一直在后面把人推着，碌碌无为的一年。心智也无任何成长，职业生涯上也没有任何突破。

华为OD的那四个月，对技术上对我锻炼很大，我上手了很多软件工程上的东西，如性能优化、软件选型。也替换了大厂的工作流，可惜因为心态，以及工作强度原因，没有坚持住。

说到心态，2020年整体心态确实很浮躁，浮躁的时候就在不断这么内心。21年，不仅要拥有一个有韧劲的心态，还要在心性上不断打磨自身。

20年学了一些web后端的知识，如MySQL、Sping等，这些对自身成长都有一定的帮助。21年也要加油。
