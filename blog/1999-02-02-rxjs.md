## 关于 Rxjs 的理解

Rxjs 通过观察者模式，实现消息推送以及订阅。可以在降低模块间的耦合度，在前端框架中，Angular 内部已经依赖了 Rxjs，并且基于 Rxjs+单类实现了依赖注入。在 Vue、React 的状态机中，都可以看到 Rxjs 的影子。

## 核心概念

| 对象         | 描述                                                                         |     |
| ------------ | ---------------------------------------------------------------------------- | --- |
| Observable   | 可观察对象，表述未来发生的值或事件的集合，数据的生产者                       |     |
| Observer     | 观察者，处理值或事件的回调函数集合，数据的消费者                             |     |
| Subscription | 订阅，入参是观察着，返回值是一个对象，可以取消观察，将生产者与消费者绑定起来 |     |
| Operators    | 操作符，对可观察对象，进行流式操作，参考数组的 map, forEach, filter 等方法   |     |
| Subject      | 一类特殊的可观察对象，可以多播                                               |     |
| Schedulers   | 调度器                                                                       |     |
