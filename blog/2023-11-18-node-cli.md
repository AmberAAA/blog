# 使用node编写脚本


使用node编写脚本需要注意以下事项，文件开头声明
```sh
#!/usr/bin/env node
```

在`package.json`中声明
```json
{
  "bin": {
    "blog": "./src/index"
  }
}
```

全局引用
```
npm link
```

当使用`pnpm`时，
```sh
pnpm link --global
```

## 必要的工具

有两个常见的包，可以让编写脚本更加刚方便
1. `commander` 构建脚本框架
2. `inquirer` 问询式架构 如常见的各种脚手架的问询
